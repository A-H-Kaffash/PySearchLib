# PySearchLib

A lightweight Python search library supporting multiple search strategies.

## Features
- Wildcard search (`*`, `?`)
- Linear search
- Binary search (exact match)
- Prefix search
- File system search
- Simple benchmarking tools

## Usage

Run the demo:

```bash
python -m examples.demo
```

Project Structure

core/ – matching logic

algorithms/ – search algorithms

benchmark/ – performance measurement

utils/ – helpers and datasets

examples/ – runnable demos
